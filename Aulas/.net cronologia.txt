2 o q é, como e onde usar .NET


Infraestrutura para desenvolvimento de software criada pela Microsoft

Uma aplicação .NET é desenvolvida para e roda uma das seguintes implementações do .NET:
.NET core
.NET Framework
Mono
Universal Windows plataform(UWP)

cada implementação inclui um ou mais >NET Runtimes (ambiente de execução):
.NET Core: CoreClr e CoreRT
.NET Framework: CLR
Mono: Mono Runtime
UWP: >NET Native 

Atualmete a Microsoft desenvolve e suporta 3 linguagens para .NET

F#
